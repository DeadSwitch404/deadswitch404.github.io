Whisper Engine 0.1.0: Incremental Builds and SHA256 Integrity


The silence grows stronger.

Whisper Engine has reached version 0.1.0.
This release brings two critical upgrades for those who build in stealth:

  • Incremental Builds
    No more full regeneration on every change.
    The engine detects modified Org files and updates only what matters.
  • SHA256 Integrity
    Every file is checked with a cryptographic hash to ensure integrity and precision.
    This means faster deploys, less noise, and more control.

Why It Matters

Whisper Engine is designed for operators who demand minimalism and predictability.
Version 0.1.0 is a foundation for the next phase: silent automation at scale.

Change Highlights

  • SHA256-based file hash building for integrity and speed
  • Incremental site generation with cache management
  • Polished pagination and DOM cleanup
